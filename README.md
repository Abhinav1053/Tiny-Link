# Tiny-Link
Tiny Link is a web app made for the assignment of Aganitha Cognitive Solutions.
This project is made with Express.js and Node.js using Neon DB as a free database and it is hosted on render.
Users can paste their long links in the website and can create their very own short link or can also make their custom chort link just like bit.ly.
